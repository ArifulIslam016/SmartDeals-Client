
# Smart Deals
Live link : [https://dragonnews-12df3.web.app/catagories/1](https://smart-deals-6ccb3.web.app/)


**Project Overview:**  
Smart Deals is an online marketplace platform designed to help users sell their old,unused or new products easily. Users can create posts for their items(Post Not implimented yet), set a min and max price, and allow other users to place bids, making the selling process interactive and competitive. The platform ensures a seamless experience with real-time bidding, clear product listings, and easy communication between buyers and sellers.
